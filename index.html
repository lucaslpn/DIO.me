O que é HTML?

Voltar

Denominada Linguagem de Marcação de Hipertexto, é uma linguagem de estruturação de textos e elementos que compõe uma página web e aplicações. É conhecida como uma linguagem de marcação que serve para escrever o documento,e por isso não é considerada uma linguagem de programação, visto que sua única função é a criação e estruturação de seção, além de parágrafos e links através do uso de tags, atributos e elementos.

Usando o Inspetor de Elementos

Voltar

Para quem trabalha com o Desenvolvimento Web, a ferramenta Inspetor de Eelementos é muito utilizadas pelos programadores no dia a dia, e ela é utilizada nos próprios navegadores. Com essa ferramenta, é possível visualizar as classes CSS, identificar e alterar os elementos de uma página, fazer simulações em sites responsivos, como para dispositivos móveis tais como smarthphones e tablets de diversos tamanhos, e por fim, é uma ótima opção para estudos

Para abrir essa ferramenta, basta usar o botão "F12" do seu teclado, ou ir nos 3 pontinhos do navegador, ir em "Mais ferramentas", logo em seguida ir em "Ferramentas do desenvolvedor", ou cliclar com o botão direito do mouse e ir em "Inspecionar". Essa ferramenta é muito utilizada para identificar erros e bugs na página web, e em diversos navegadores como o Chrome, Firefox e Safari disponibilizam esse recurso de inspecionar elementos.

Estrutura básica do HTML

Voltar

A estrutura básica do HTML começa pela seleção de tags, tais como o "html", "head" e "body", a partir da instrução "!DOCTYPE"

A partir dessas tags, é possível dá início de uma página web com html, pois o escopo da etrutrura html é dada pela declaração de início do html, a cabeça e o corpo, e todas essas tags ficam entre os sinais de de maior e menor.

Falando sobre Tags

Voltar

Como funcionam as tags que compõe a estrutura básica do HTML?

  • html: É a tag que estrutura o documento index.html, pois essa tag antecede todo o conteúdo no escopo da estrutura, e logo em seguida serão usadas as tags "head" e "body" que são tags irmãs.

  • head: Essa tag armazena as informações sobre o ducomento HTML, onde é especificada a tag title dentro dela, que é a barra de título da página, pois sem ela, não é possível documentar a página como HTML. É utilizada o charset dentro da tag meta, com o valor UTF-8 ou chamado de Unicode, que determina o idioma usado na página.

  • body: A tag body é usada para conter o corpo do documento HTML, na qual é todo o conteúdo visível no escopo da página, onde será as diversas tags que compõe o conteúdo.

  • i: Deixa o texto em itálico

  • strong: Deixa o texto em negrito.

  • p: Cria um parágrafo para determinada seção.

  • input: Abre uma caixa de texto.

  • img: Possobilita colocar uma imagem na página.

Atributos das Tags

Voltar

Há atributos próprias e genéricas que são utilizadas nas tags.

Exemplo desses atributos:

  • "id": É um identificador de determinado elemento, muito utilizado ao usar JavaScript.

  • "style": Utilizado para usar no CSS, pois nele é usado comandos para formatar o HTML, ou seja, estilizar o conteúdo.

  • "class": É usado para padronizar determinado conteúdo pré-definido

  • "type": Determina o que pode ser contido dentro da caixa do input

  • "src": Usado para indexar imagens através de links na web, é o caminho da imagem

  • "width": Atributo de css que determina o tamanho da imagem

Textos

Voltar

Dentro do elemento body, é onde serão colocados os textos, e com isso, podemos definir os texto por seus cabeçalhos, que são representados na tag "h", que é heading, que são recursos utilizados para destacar títulos e subtítulos da página web, e por isso é representado como "h1", "h2", "h3"..

O h1 tem uma margem maior, por ser o título maior e em negrito, e os outros h que são enumerados, vão diminuindo cada vez mais o tamanho.

É possível colocar observações, citações em um texto, e é usada a tag "blockquote" e dentro dela a tag "i", na qual dá um espaçamento no texto, e por isso ele se desprende do texto principal.

Outra tag que pode ser utilizada nos textos, que serve para sublinhar algo que pode se destacado, é com a tag "u" ao redor da palavra ou frase. Assim como a tah "mark" que destaca uma cor de fundo na palavra ou frase de destaque. Ou então para deixar alguma palavra, letra ou número como forma de raiz quadrada, pode ser usada a tag "sup" como notinha de rodapé.

Listas ordenadas e não ordenadas

Voltar

As listas ordenadas e não ordenadas são representadas de maneira fácil e lógica, as ordenadas são representadas por uma bolinha preta, chamada de bullet, e as não ordenadas são representadas por números. Mas todas elas devem ser representadas pela tag "li" para se tornar uma lista de fato.

Listas ordenadas: "ol"

Listas não-ordenadas: "ul"

Links

Voltar

Os links no HTML são determinados pela tag "a", nas quais são colocadas o atributo "href", que significa Hypertext Reference, nas quais são o destino do link endereçado. Tudo que é colocado na tag "a" sevre como um redirecionador, isto é, ao ser clicado, servirá a função de ir para algum endereço marcado dentro do atributo "href".
